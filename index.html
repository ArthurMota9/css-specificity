How does the browser define the priority of selectors?

Selectors List

  • (*) Universal selector;
  • (div) element selector;

(*) Universal selector

The * selector selects all elements.

(*) Universal selector


							* {
						    background-color: yellow;
							}

The * selector selects all elements.

(*) Universal selector


							div * {
						    background-color: yellow;
							}

Select all elements inside <div> elements

(div) Element selector

The element selector selects all elements with the specified element name.

(div) Element selector


							div {
						    background-color: yellow;
							}

Select and style all <div> elements

(div, p) Element selector


							div, p {
							  background-color: yellow;
							}

Select and style all <div> and <p> elements

(div p) Element selector


							div p {
							  background-color: yellow;
							}

Selects all <p> elements inside <div> elements

(.myClass) Class selector

The class selector selects elements with a specific class attribute.

(.myClass) Class selector


							.myClass {
							  background-color: yellow;
							}

Select and style all elements with class="myClass"


							<div class="myClass"> My yellow background </div>
							<p class="myClass"> My yellow background </p>

(p.myClass) Class selector


							p.myClass {
							  color: white;
							}

Style all <p> elements with class="myClass" 


							<p class="myClass"> My white text </p>

(.myClass .myClass2) Class selector


							.myClass .myClass2 {
							  color: white;
							}

Selects all elements with class="myClass2" that is a descendant of an element with class="myClass" 


							<div class="myClass">
							  <p class="myClass2"> My white text </p>
							</div>

(#myId) ID selector

The id selector styles the element with the specified id

(#myId) Class selector


							#myId {
							  color: yellow;
							}

Style the element with id="myId" 


							<p id="myId"> My yellow text </p>

inline style

The HTML Style Attribute

(style="color: red") inline style


							<p style="color: red"> My red text </p>

!important

When an important rule is used on a style declaration, this declaration overrides any other declarations.

!important


							.myClass {
							  background-color: yellow !important;
							}
	
							.myClass {
							  background-color: red;
							}

!important

							  background-color: yellow !important;
							}
	
							.myClass {
							  background-color: red;
							}

What is Specificity?

If there are two or more conflicting CSS rules that point to the same element, the browser follows some rules to determine which one is most specific and therefore wins out.

CSS Specificity

Selector Value Description
* 0 - 0 - 0 - 0 Universal selector
div 0 - 0 - 0 - 1 1 element
.myClass 0 - 0 - 1 - 0 1 class
#myId 0 - 1 - 0 - 0 1 attribute id
inline style 1 - 0 - 0 - 0 attribute HTML style
